#### 未完成

* 和弦只支持简单分解和弦
* 不支持副旋律
* 生成文件只支持钢琴音色

#### 待定

* 除了音色、音调、速度、长度外，其他不能设置，暂时也不打算添加，可配置的越多就越复杂，增加参数的复杂度还不如直接学习作曲
